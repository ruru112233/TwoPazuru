内容物は以下のとおりです。
【商 品 名】ウィンドウ＆ＵＩパーツ素材セット２
【価　　格】４００円＋税
【収録形式】32bit PNGファイル
【収 録 数】基本：バリエーション込み１４０種類(基本)
　　　　　　ver1.1追加：３７種類＋ツクール向け１１種類

こちらはWOLF RPGエディター(http://www.silversecond.com/WolfRPGEditor/)の
ピクチャ機能「お手軽ウィンドウ」での使用を想定した素材集です。
基本システムでの動作は未確認です。予めご了承ください。
仕様が合えば他の開発ツールはもちろん、媒体問わず創作全般にお使いいただけます。

【使い方】
一部の「omake」で始まるファイルを除いて全てお手軽ウィンドウ対応です。
最大サイズは上限なし(一部大きすぎると見た目が崩れるものがあります)、
最小サイズは画像サイズの2/3までとなります。
例外として「ボタンA」「プレートB,C」「両サイドの矢印」「おまけH」のみ縦＝画像サイズでの使用を推奨しています。
「プレートミニA」のみ縦サイズ12〜18での使用を推奨しています。
ライン系の素材は縦が伸縮しないよう上に固定してありますので、中央指定の場合少々ずれます。
サンプルデータは非暗号化です。組み方の参考、一部流用などご自由に。ただしソースは汚いです。

（追記）アップデートによりRPGツクールMV対応素材を追加しました。
枠、矢印画像がメインです。ウィンドウ部、テクスチャ、文字色などはイメージの合うものに調整してください。
【使い方】
プロジェクトフォルダにある「Window.png」と置き換えてください。
※置き換え前にかならずバックアップをお取り下さい。
※こちらは「RPGツクールMV　Ver.1.5.1」でのみ動作を確認済みです。
